配置方式

vue2 配置

  1. 安装 pinia
    • npm i pinia -S
  2. 安装 @vue/composition-api
    • npm i @vue/composition-api -S
  3. 打开入口文件 main.js
    1. 引入 pinia 和 @vue/composition-api
      1. import { createPinia, PiniaVuePlugin } from 'pinia'
      2. import VueCompositionAPI from '@vue/composition-api'
    2. 注册插件 PiniaVuePlugin 和 VueCompositionAPI
      1. Vue.use(PiniaVuePlugin)
      2. Vue.use(VueCompositionAPI)
    3. 创建 pinia 实例（创建根存储，传递给应用程序）
      • const pinia = createPinia()
    4. Vue 实例化配置项中添加 pinia
      • {pinia: pinia}
  4. src 目录下，创建 store 目录
    • 目录下创建 index.js 文件
      1. 引入 pinia
        • import { defineStore } from 'pinia'
      2. 调用 defineStore
        1. 参数1：容器ID，必须唯一
        2. 参数2：配置对象
          1. state：函数
            1. 存储组件状态（数据）
            2. 返回值为对象，定义初始值
          2. getters：对象
            1. 计算属性
          3. actions：对象
            1. 封装业务逻辑，修改 state
      3. 暴露 difineStore 函数调用返回值

项目如果是使用 Vue CLI 构建，调用命令 vue add vue-cli-plugin-pinia ，使用第三方插件，一步完成上述所有操作

vue3 配置

  1. 安装 pinia
    • npm i pinia -S
  2. 打开入口文件 main.js
    1. 引入 pinia
      • import { createPinia } from 'pinia'
    2. 注册插件（创建根存储，传递给应用程序）
      • app.use(createPinia())
  3. src 目录下，创建 store 目录
    • 目录下创建 index.js 文件
      1. 引入 pinia
        • import { defineStore } from 'pinia'
      2. 调用 defineStore
        1. 参数1：容器ID，必须唯一
        2. 参数2：配置对象
          1. state：函数
            1. 存储组件状态（数据）
            2. 返回值为对象，定义初始值
          2. getters：对象
            1. 计算属性
          3. actions：对象
            1. 封装业务逻辑，修改 state
      3. 暴露 difineStore 函数调用返回值

store 说明

  1. store 它托管全局状态。它有点像一个始终存在并且每个人都可以读取和写入的组件
  2. store 实例无法解构赋值
  3. 引入 pinia 的 storeToRefs 函数，调用此函数可以对 store 实例进行结构赋值
    1. import { storeToRefs } from 'pinia'
    2. const { name, doubleCount } = storeToRefs(store实例)

definestore 函数

  1. definestore 函数用来生成一个可以实例化 store 的函数
  2. 实参
    1. 参数1：store ID，作为 store 标识
    2. 参数2：配置对象
  3. 返回值：用来实例化 store 的 函数

store 配置对象

  1. state：函数
    1. 存储全局状态（数据）
    2. return 返回对象中定义初始值
  2. getters
  3. actions
    1. 用于定义修改 state 中数据的函数
    2. 引用 state 数据，用 this 引用

使用方式

Vue2

  1. 引入自定义的 store 模块
    • import {useCounterStore} from "../store"
  2. 组件中引入函数 mapState 和 mapActions
    1. import { mapState, mapActions } from "pinia"
  3. 组件 computed 配置项中调用 mapState
  4. 组件 methods 配置项中调用 mapActions
  5. 直接调用 computed 中展开的数据，访问 store 中数据
  6. 调用 methods 展开的函数，修改数据
