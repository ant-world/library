简介

什么是TypeScript

TypeScript 是 JavaScript 的超集，其包含了 JavaScript 基本语法也具有额外扩展

搭建 TypeScript 环境

通过 npm 安装

  1. 下载 node.js https://nodejs.org/zh-cn/
  2. 安装 node.js (直接下一步)
  3. 安装 TypeScript
  4. npm installl typescript -g

查看 TypeScript 版本

tsc -V

编译 .ts 文件

  1. 创建 .ts 文件
  2. 命令行执行 tsc xx.ts 将 .ts 文件编译为 .js 文件

TypeScript 注意事项

  1. TypeScript 无法被浏览器直接使用，需要通过编译器转换成 JavaScript 才能被浏览器使用
  2. HTML 页面可以直接引入 .ts 文件，如果 .ts 文件中只有纯 JavaScript 语法，在谷歌浏览器中也可以正常运行。但如果有 TS 语法将无法运行。
