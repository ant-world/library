VSCode操作

发表于2022-06-23|更新于2022-09-02|工具
|阅读量:

命令窗口

打开命令窗口

Ctrl + ~

无法执行命令

  • vscode使用的是Powershell，Powershell对于不受信任的程序禁止执行，设置执行策略即可
    1. 管理员身份打开 Powershell
    2. 查看执行策略：get-ExecutionPolicy
      1. 受限制状态：Restricted
      2. 远程签名：RemoteSigned
    3. 设置执行策略为【远程签名】：set-ExecutionPolicy RemoteSigned
    4. 选择 y

快捷键

注释

单行注释

ctrl + /

多行注释

ctrl + shift + /

功能开启

MarkDown 实时预览

  1. F1
  2. 搜索 markdown
  3. 打开侧边预览
