ProxmoxVE虚拟机配置

发表于2022-07-12|更新于2022-07-12|服务器
|阅读量:

创建虚拟机

  • 系统
    • 选择默认
  • 磁盘
    • 总线/设备
      • VirtIO Blok
        • 需要相关 VirtIO 驱动ISO镜像
        • 驱动地址：https://fedorapeople.org/groups/virt/virtio-win/direct-downloads/archive-virtio/
      • IDE
        • 不需要相关ISO驱动镜像
  • CPU
    • 核心
      • windows系统：2核
      • linux系统：1核
    • 类别
      • host
  • 内存
    • windows：2048
    • linux：1024
  • 网络
    • 模型：VirtIO(半虚拟化)

上传系统/驱动镜像

点击 local 从控制机进行镜像的上传

外接USB设备

  1. 选中创建的虚拟机
  2. 硬件
  3. 添加
  4. USB设备
  5. 使用USB供应商/设备ID

安装系统

安装系统时，弹出使用 CD/DVD 启动，需要点击任意键盘按键。否则启动网络引导

命令

强制关闭虚拟机

qm stop 虚拟机ID

查看IP地址

ifconfig

需要先下载 ifconfig 命令工具

apt-get install net-tools

如果无法下载，需要先进行更新 apt-get 相关数据，执行更新后再重新下载

apt-get update
